Правила клуба (Retro-List):

  • Никогда не пушить в мастер без ревью.
  • Комментарии в коде обязательны.
  • Соблюдать PEP-8, даже если никто не видит.
  • Кофе — топливо для кода.

Вот пример кода нашего типичного участника:

class Student(Programmer):
    def __init__(self):
        self.coffee_level = 100
        self.knowledge = 0
    
    def learn_django(self):
        while self.coffee_level > 0:
            self.read_docs()
            self.write_code()
            self.coffee_level -= 10
            self.knowledge += 1
