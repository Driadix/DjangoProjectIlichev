## 2. МАТЕРИАЛЫ, КОТОРЫЕ НЕОБХОДИМО ПОДГОТОВИТЬ (Графика и схемы)

1.  **Карта сайта (раздел 2.1)** — древовидная структура страниц (Главная -> Каталог -> Категория -> Товар и т.д.).
2.  **Схема базы данных (раздел 2.2.1)** — прямоугольники таблиц со связями 1-ко-многим (как в Приложении Б методички).
3.  **Скриншоты (раздел 2.3)** — красивые скрины: Главной, Каталога, Корзины, формы входа в стиле "ретро-терминала".
4.  **5 Блок-схем (раздел 3)** — по ГОСТ 19.701. Основные элементы:
    *   *Овал (терминатор)* — Начало/Конец.
    *   *Параллелограмм (данные)* — Ввод/вывод (например, получение POST-данных).
    *   *Ромб (решение)* — Условия (Авторизован ли? Данные валидны?).
    *   *Прямоугольник (процесс)* — Действие (Сохранить в БД, сменить статус).

## 3. ПРАВИЛА ОФОРМЛЕНИЯ ТЕКСТА (Памятка ГОСТ и ЕСПД для Word)

*   **Шрифт:** Times New Roman 14 пт.
*   **Интервал:** 1.5, выравнивание по ширине.
*   **Поля:** Верхнее 3 см, Нижнее 2 см, Левое 2.5 см, Правое 1.5 см.
*   **Отступ первой строки (абзац):** 1 см (10 мм).
*   **Номера страниц:** Внизу по центру.
*   **Заголовки:** Каждый раздел с новой страницы. Точка в конце заголовка НЕ ставится!
*   **Рисунки:** Ориентированы по центру. Подпись внизу по центру, без точки в конце. 
    *   *Пример: Рис. 2.1. Название рисунка*
*   **Таблицы:** Слово "Таблица 2.1" в правом верхнем углу. Название таблицы ниже по центру.
*   **Ссылки:** Обязательны ссылки на рисунки `(рис. 2.1)` и таблицы `(таблица 2.1)` в тексте.
